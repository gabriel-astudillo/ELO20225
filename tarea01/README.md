# ELO20225 Tarea 1

## Objetivos

* Implementar código modular a partir de una solución monolítica. 
* Utilizar tipos de datos `enum` para mejorar la legibilidad del código.
* Realizar mediciones de tiempo de ejecución en un código sencillo con el fin de evidenciar
las diferencias que existe entre usar la memoria del sistema y un sistema de Entrada/Salida.

## Descripción

Esta tarea consiste implementar codigos sencillos en C para concretizar conceptos de dicho lenguaje vistos en clases. El trabajo consiste en dos partes. La primera es ejercitar el concepto de programación modular, mientras que la segunda se enfoca en realizar mediciones de tiempo de ejecución. En ingeniería, y en particular en el campo de la ingeniería electrónica, se debe tener clara la idea que para analizar el comportamiento de un sistema se deben medir sus características principales. En el caso de un programa, una de esas variables de interés es el tiempo de ejecución.

## Materiales a utilizar

Para este trabajo, se utilizará VSCode como entorno de desarrollo. Se recomienda tener instalado algún plugin que permite visualizar archivos `Markdown`, además de los módulos para compilar programas en C. Estos módulos o plug-ins están descritos en el Aula Virtual de la asignatura.
